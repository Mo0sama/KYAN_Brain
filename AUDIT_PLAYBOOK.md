# KYAN Audit Playbook

Use this sequence for every client:

1. Start with the Google Form audit.
2. Review the client page/profile/website manually.
3. Score the 9 audit areas from 0 good to 5 critical.
4. Identify the biggest bottleneck.
5. Recommend one first paid step.
6. Give top 5 fixes.
7. Avoid guarantees.

## Core Tools

- Facebook and Instagram: public profile view, Meta Business Suite, Insights, Inbox, WhatsApp Business
- Website: mobile browser, Google PageSpeed Insights, Google Search Console, Google Analytics if available
- Local presence: Google Business Profile dashboard and public Maps listing
- CRM/follow-up: WhatsApp chats, existing sheets, order/payment examples, manual process map
- Automation: n8n, Google Forms, Google Sheets, notification channel, sample data
- Technical: hosting panel, DNS panel, SSL checker, backups, screenshots

## Audit Order

1. Clarity: can the customer understand the offer?
2. Trust: does the business look real and safe?
3. CTA: is the next action clear?
4. Content: is the content useful and consistent?
5. Follow-up: are leads tracked?
6. Orders/payments: are operations tracked?
7. Website: does the page convert?
8. Technical: are there blockers?
9. Automation: is the process ready?

## Decision Rules

- If clarity/trust/CTA are weak, start with Facebook/Instagram/Page Setup.
- If messages arrive but sales are low, audit follow-up and CRM.
- If orders/payments are messy, build Google Sheets CRM before automation.
- If process is unclear, do not automate yet.
- If website buttons/forms/SSL are broken, technical support comes first.
- If content is random but page is clear, start content planning.
- If multiple systems need ongoing work, recommend monthly support after the first delivery.
